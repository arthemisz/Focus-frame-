# 📷 FocusFrame Media — Portfolio Website

## 🌐 Overview
**FocusFrame Media** is a responsive multi-page portfolio website designed for a professional photography and videography team. The website showcases their creative work, services, and brand identity with a modern editorial-style aesthetic.

This project was developed as part of the **Nexus Front-End Boot Camp Project**, focusing on UI/UX design and front-end development using **Figma, HTML, and CSS**.

---

## 🎯 Objective
The goal of this project is to simulate a real-world client scenario by:

- Designing a visually appealing portfolio in **Figma**
- Translating the design into a fully responsive website
- Applying **UI/UX principles** and **design systems**
- Building clean and structured code using **HTML & CSS only**

- - - 

## 🛠️ Tech Stack

- **Design:** Figma  
- **Frontend:** HTML5, CSS3  
- **Fonts:** Google Fonts  
- **Icons:** SVG format
